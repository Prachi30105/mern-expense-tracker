## ✨ Features

- Add, edit, and delete expenses
- Expense categories (Food, Travel, Bills, etc.)
- Dashboard with analytics charts
- User authentication (JWT)
- Monthly expense tracking
- Export expenses report

---

## 🛠 Tech Stack

**Frontend**

- React
- CSS / Tailwind
- Axios

**Backend**

- Node.js
- Express.js
- JWT Authentication

**Database**

- MongoDB
