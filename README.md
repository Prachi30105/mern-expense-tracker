# mern-expense-tracker
# 💰 MERN Expense Tracker

A **full-stack Expense Tracker application** built using the MERN stack that allows users to manage and analyze their daily expenses efficiently.
Users can add, view, filter, and track expenses across different categories with a clean and responsive interface.

# ✨ Features

* Add new expenses
* Edit or delete expenses
* Expense categories (Food, Travel, Bills, etc.)
* View all expenses in a structured list
* Dashboard with expense analytics
* User authentication and authorization
* Filter expenses by category or date
* Secure backend API

# ⚙️ Installation and Setup

Clone the repository

git clone https://github.com/Prachi30105/mern-expense-tracker.git

Navigate to the project directory

cd mern-expense-tracker

---

### Install Backend Dependencies

cd backend
npm install

Run backend server

npm start

---

### Install Frontend Dependencies

cd frontend
npm install

Run frontend

npm start

---

# 🔐 Environment Variables

Create a `.env` file in the backend folder and add:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
